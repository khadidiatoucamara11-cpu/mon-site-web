PROJET : Site web Institut Polyvalent du Futur
GROUPE 4 - Développement Web L2 Semestre 1

COMMENT OUVRIR LE SITE EN LOCAL :

Méthode 1 (Recommandée) :
- Utiliser Live Server dans VS Code

STRUCTURE DU SITE :
- index.html : Page d'accueil (présentation, histoire, chiffres clés)
- formations.html : Liste des formations avec filtre par domaine
- enseignants.html : Grille des enseignants avec fiches détail
- contact.html : Formulaire de contact et coordonnées

TECHNOLOGIES UTILISÉES :
- HTML5 sémantique
- CSS3 (charte graphique, responsive, animations)
- JavaScript (chargement XML, DOM manipulation)
- XML (stockage données formations et enseignants)
